## App Store Description
AITRADER is a native iPhone app for monitoring the AITRADER trading dashboard.

It is built for disciplined trading operations, with a focus on live execution visibility, market context, strategy review, and comparison between actual trades and simulated results.

Key usage areas:

- live session status and control
- KRX market monitoring and candidate review
- actual transaction review
- simulation and backtest inspection
- live versus simulated result comparison

AITRADER connects to a reachable AITRADER server environment. It is intended for private or internal operational use rather than consumer brokerage onboarding.
